Instructions on how to set up your phone to report it's position to the live tracking system.

Configuration instructions

  1. Install GPS Logger.
  2. Open settings menu from the top left.
  3. Select OpenGTS Setup.
  4. Click Enable OpenGTS.
  5. Click Enable Automatically Send.
  6. Click Server and write SERVER NAME.
  7. Click Port and write SERVER PORT.
  8. Click Communication Method and select HTTP GET.
  9. Click Server Path and write /gps.
  10. Click Device ID and write a descriptive name.
  11. Go back to the main page and start the tracking with the large play icon.

Please note

  • If no position is reported in four hours the marker will be removed.
  • The Device ID will be used as name, please write something short but descriptive.
  • Clicking a marker selects it and shows the coordinates and last update time in a box at the bottom. It also shows centers the map on each update of that device.
